# Sistema de Gerenciamento de Endereços

Projeto desenvolvido em ASP.NET Core MVC para gerenciamento de endereços com autenticação de usuários e integração ViaCEP.

## Funcionalidades

- Login de usuários
- CRUD de endereços
- Busca automática por CEP
- Exportação CSV

## Tecnologias

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- ViaCEP API

## Como executar

1. Configurar connection string
2. Executar scripts SQL
3. Executar projeto
